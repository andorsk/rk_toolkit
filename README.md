# RK Toolkit

Standard package and toolkit for RK Diagrams.

#### Installation
``` sh
python -m pip install .
```
##### Running Tests

``` sh
pytest -m .
```

### Test Coverage Report

``` sh
python -m pip install pytest pytest-cov
pytest --cov=. .
```
